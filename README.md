# NervoSound – Music Therapy Web App

A simple web-based project that plays selected frequency tracks (432Hz / 852Hz / 963Hz), tracks listening duration, and logs self-reported mood changes to generate basic analytics.

## Features
- Audio playback for multiple frequencies
- Listening time tracker (per frequency)
- Session logging:
  - Participant name
  - Frequency used
  - Mood before/after (0–10)
  - Duration (seconds)
  - Improvement calculation (after - before)
- Analytics summary (total sessions, average improvement, most used frequency)
- Data persistence using LocalStorage

## Tech Stack
- HTML / CSS / JavaScript
- Hosted with GitHub Pages

## Notes
This project records self-reported mood ratings and is intended for educational purposes only.
